# 性能优化实战项目

## 项目目标
在实际业务中完成一次完整的性能优化，产出量化的优化报告。

## 优化清单

### 加载性能
- [ ] 代码拆分（Route-based + Component-based）
- [ ] Tree Shaking 验证
- [ ] 图片优化（WebP/AVIF + 响应式图片）
- [ ] 关键 CSS 内联
- [ ] 预加载关键资源（preload/preconnect）
- [ ] 服务端渲染（SSR/SSG/ISR 选型）

### 运行性能
- [ ] 虚拟列表（万级数据渲染）
- [ ] 防抖/节流 优化交互
- [ ] Web Worker 处理重计算
- [ ] requestAnimationFrame 优化动画
- [ ] React.memo / useMemo 减少重渲染

### 衡量指标
| 指标 | 优化前 | 目标 | 优化后 |
|------|--------|------|--------|
| LCP | - | < 2.5s | - |
| INP | - | < 200ms | - |
| CLS | - | < 0.1 | - |
| JS Bundle | - | < 200KB | - |
| 首屏API数 | - | ≤ 3 | - |

## 学习产出
- [ ] 完成性能基线测量
- [ ] 实施 5+ 项优化措施
- [ ] 撰写优化报告（含数据对比图表）
- [ ] 团队分享（PPT）
