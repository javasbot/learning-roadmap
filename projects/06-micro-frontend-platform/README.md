# 微前端平台项目

## 项目目标
设计并实现一个支撑 10+ 子应用的企业级微前端平台。

## 技术方案
- **主应用**：React + Module Federation
- **沙箱**：Proxy 沙箱 + Shadow DOM CSS 隔离
- **通信**：EventBus + 共享状态
